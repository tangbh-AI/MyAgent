## 规则

1. **说中文** — 始终用中文回复，代码注释也写中文。
2. **先计划** — 收到任务先制定计划，获得确认后再执行。
3. **逐步汇报** — 每一步操作都向用户报告，有错误立即说。
4. **文档同步** — 每次代码变更同步更新 `README.md` 和 `PROGRESS.md`。
5. **用 ccuse** — 所有 Python 操作必须在 conda 环境 `ccuse` 中执行。
6. **锁定本文件** — 除非用户明确同意，绝不修改此文件。
7. **保留模型切换** — 用户可随时通过 `config.yaml` 或 `--model` 切换 AI 模型。

# ---------------------------------------------------------------------------
# 示例 1: 悬臂梁静力分析（入门）
# ---------------------------------------------------------------------------
分析一个悬臂梁，长1米，矩形截面50mm宽100mm高，钢材料。一端完全固定，自由端受到向下1000N的集中力。做静力分析，网格尺寸10mm。

# 预期追问: 无（参数完整）
# 预期结果: 最大应力、最大位移、安全系数


# ---------------------------------------------------------------------------
# 示例 2: 平板模态分析
# ---------------------------------------------------------------------------
分析一块铝板，尺寸500x300mm，厚度5mm。四边完全固定。做模态分析，提取前6阶固有频率和振型。

# 预期追问: 材料是否使用预定义铝材 / 网格尺寸
# 预期结果: 各阶频率、振型图


# ---------------------------------------------------------------------------
# 示例 3: 轴扭转分析
# ---------------------------------------------------------------------------
分析一根钢轴，直径50mm，长度500mm。一端完全固定，另一端施加500Nm的扭矩。计算应力分布和扭转角。

# 预期追问: 网格尺寸 / 是否需要非线性
# 预期结果: 最大剪应力、扭转角


# ---------------------------------------------------------------------------
# 示例 4: 压力容器分析
# ---------------------------------------------------------------------------
分析一个圆柱形压力容器，内径400mm，壁厚10mm，长度1000mm。两端封头为半球形。材料为钢。内部施加2MPa的均匀压力。做静力分析。

# 预期追问: 网格尺寸 / 是否需要对称简化
# 预期结果: 环向应力、轴向应力、最大位移


# ---------------------------------------------------------------------------
# 示例 5: 热应力分析
# ---------------------------------------------------------------------------
分析一根两端固定的钢杆，长度500mm，圆形截面直径20mm。初始温度20°C，均匀加热到200°C。分析热膨胀产生的热应力。

# 预期追问: 热膨胀系数 / 网格尺寸
# 预期结果: 轴向热应力、约束反力
